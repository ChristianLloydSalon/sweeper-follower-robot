# Sweeper Follower Robot
A robot with a garbage can that follows a street sweeper

# Electronic Components Used
* HC-05 Bluetooth Module
* NEO 6m GPS Module
* HMC5883L Compass
* L298N Motor Driver
* Arduino Mega 2560
* Motors
